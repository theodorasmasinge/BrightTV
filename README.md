# BrightTV
# Overview of What Was Done
The goal of the BrightTV case study was to analyze user viewing behavior to identify patterns in content consumption, peak viewing periods, and channel performance. The analysis aimed to help BrightTV improve engagement, increase viewership, and guide strategic decisions around programming and user acquisition.
# How i did the case study
1. Removed unnecessary characters and converted timestamps into correct date/time formats.
2. Extracted:Day of week , Hour of Day,Duration watched and viewer consumption
3. Created additional fields for deeper analysis: Day of week , Hour of Day,Duration watched , ViewTime
4. Built pivot tables in Excel for: Viewership by weekday, Viewership by Race, Viewership By province, Viewership Per Age_group, Total Minutes watched by Month Name, Total Minutes Watched By Gender
   # Insights Found in the case study
   1. Channel performance
   2. Time-of-Day Consumption
   3. Day-of-Week Trends
   4. Gender Split
